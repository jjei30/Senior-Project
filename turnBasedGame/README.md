## Turn Based Game

This is a senior project for COS which will focus on Turn Based RPG Game that includes AI opponents with respect to decision evaluation.

## Map

The Map uses a form of Wave Function Collapse (WFC) to generate tiles that would correspond with each other

The letters correspond to the following tiles:
- X = Grass
- Y = Tree
- M = Mountain
- S = Sea
